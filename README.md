# Personal Pokedex

personal pokedex is a web application which pulls information on pokemon from the pokemon API using an ajax call





## Usage

```javascript
class Pokemon{
    constructor( hp, attack, defense, ability){
        
        this.hp=hp;
        this.attack=attack;
        this.defense=defense;
        this.ability=ability;
    }
}
```
